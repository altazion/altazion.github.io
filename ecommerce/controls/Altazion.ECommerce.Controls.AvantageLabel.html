Class AvantageLabel

Affiche un span avec le code avantage actif sur le panier en cours

Héritage
System.Object
System.Web.UI.Control
System.Web.UI.WebControls.WebControl
System.Web.UI.WebControls.Label
AvantageLabel
Syntaxe
public class AvantageLabel : Label, IComponent, IDisposable, IParserAccessor, IUrlResolutionService, IDataBindingsAccessor, IControlBuilderAccessor, IControlDesignerAccessor, IExpressionsAccessor, IAttributeAccessor, ITextControl
Remarques

Vous pouvez utiliser ce contrôle dans toutes les pages du site, sous la forme :

<ecom:AvantageLabel runat="server"
  TextIfActive="Votre code avantage : {0}"
  /gt;

Si vous renseignez la propriété TextIfInactive, le contenu de cette propriété sera affiché lorsque le panier n'est pas associé à un code avantage. Si cette propriété est vide et que le panier n'a pas de code avantage actif, le <span> ne sera pas ajouté dans la page

Propriétés

TextIfActive

Obtient ou définit le texte à afficher si un code avantage saisi est actif.

Déclaration
public string TextIfActive { get; set; }
Remarques

Ce texte est utilisé pour formater le code saisi : utilisez {0} dans ce texte pour afficher le code.

TextIfInactive

Obtient ou définit le message lorsqu'aucun code n'est saisi

Déclaration
public string TextIfInactive { get; set; }

Implements

System.IDisposable
