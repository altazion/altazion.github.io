Class FilAriane

affiche un ul représentant le fil d'Ariane

Héritage
System.Object
System.Web.UI.Control
System.Web.UI.WebControls.WebControl
FilAriane
Syntaxe
public class FilAriane : WebControl, IComponent, IDisposable, IParserAccessor, IUrlResolutionService, IDataBindingsAccessor, IControlBuilderAccessor, IControlDesignerAccessor, IExpressionsAccessor, IAttributeAccessor
Remarques

La structure du fil d'ariane est la suivante :

  • un UL contient tout le fil d'ariane
  • le noeud 'acceuil' est dans un a dans un li
  • toutes les pages permettant de "remonter" sont dans a, dans des li avec un marquage rdfa
  • la page courante est dans un li simple
