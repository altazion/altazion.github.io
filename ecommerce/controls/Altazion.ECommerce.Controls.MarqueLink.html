Class MarqueLink

Affiche un lien vers la page de recherche pour une marque donnée ou vers le site de la marque

Remarques

Vous pouvez utiliser ce contrôle dans les éléments qui affichent une marque :

  • Un CPointSoftware.ECommerce.Tools.ProductPanel : dans ce cas, la marque de l'article sera prise en compte
  • Une descente produit correspondant à une marque, pour en faire, par exemple, un entête de page
  • Un CPointSoftware.ECommerce.Tools.RechercheCritereLabel ou dans le contrôle RechercheCritereMarque.ascx : pour afficher la marque d'un critère (d'un facet) de recherche
  • Un contrôle dont la source de données serait un CPointSoftware.ECommerce.Tools.MarquesSource
  • ou tout contrôle personnalisé qui effecturait un databinding sur la table catalog_marques.
