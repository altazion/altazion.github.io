Class BonsAchatsAjouterLink

Affiche un a permettant d'ajouter un bon d'achat via le code saisi dans un textbox dont le nom est fourni dans TextboxID

Remarques

Ce contrôle ne peut être utilisé que dans le cadre du contrôle personnalisé 

BonsAchatsSelectionnables.ascx

Propriétés

TextboxID

Obtient ou définit l'identifiant de la System.Web.UI.WebControls.TextBox dans laquelle sera saisi le code du bon d'achat à ajouter

Déclaration
public string TextboxID { get; set; }
Remarques

La System.Web.UI.WebControls.TextBox pointée doit être dans le même espace de nommage que ce contrôle
