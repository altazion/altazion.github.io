Commande Expediee

Cet évènement est déclenché lorsqu'une commande est expédiée (partiellement ou intégralement)

Informations sur l'évènement :

  • Catégorie : e/rp
  • Code : ExpeditionBcd
  • Classe de données : OmsCommandeExpeditionEventData

Type(s) de données :
